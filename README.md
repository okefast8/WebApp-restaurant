====================  WebApp Project  ====================

Описание проекта:
	WebApp — это веб-приложение для автоматизации работы столовой.

Система позволяет:
	Пользователям — просматривать меню, оформлять заказы, оплачивать их и управлять профилем.
	Поварам — принимать и готовить заказы.
	Администратору — управлять меню и складом.
	Системе — автоматически списывать ингредиенты со склада с учетом удалённых ингредиентов и аллергенов.

Проект реализован на Spring Boot с использованием архитектуры MVC.

====================  WebApp Project  ====================

Архитектура проекта:
	Проект построен по слоистой архитектуре:
	Controller → Service → Repository → Database

Основные пакеты:
	controller:
		auth — регистрация и авторизация
		menu — работа с меню
		orders — корзина и оформление заказа
		chef — интерфейс повара
		transactions — баланс и профиль
	service:
		CartService - работа с корзиной
		PaymentService - работа с оплатой
		UserService - работа с пользователями
		CustomUserDetailsService - работа пользователями
	repository:
		JPA-репозитории для работы с БД
		model

Реализованный функционал:
	Пользователь:
		Регистрация и авторизация
		Просмотр меню
		Добавление в корзину
		Удаление ингредиентов из блюда
		Оформление заказа
		Оплата заказа
		Просмотр истории
		Управление аллергенами
    Оставление отзывов
	Повар:
		Просмотр новых заказов
		Принятие заказа
		Автоматическое списание ингредиентов
		Учет удалённых ингредиентов
		Изменение статуса заказа (NEW → ACCEPTED → READY)
	Склад:
		Просмотр остатков
		Пополнение ингредиентов
		Контроль нехватки
